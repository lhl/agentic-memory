# Papers (Archived PDFs + Text Snapshots)

This directory contains **archived academic/industry papers** referenced by our analyses.

Conventions:
- PDFs: `arxiv-<id>.pdf` (git-lfs tracked via repo-wide `*.pdf` rule)
- Optional text snapshots: `arxiv-<id>.md` (generated via `pdftotext -layout` for grep/search)

The authoritative “what we think” summaries live under `references/*.md`. Root-level `ANALYSIS-*.md` files are critical deep dives intended for synthesis.
